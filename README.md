# Career Planning Dashboard - Job Posting Data Analysis
An interactive dashboard for analyzing job market data, salary trends, and skill requirements to help with career planning.

## 🚀 Live Demo
[Click here to view the live app](https://mysalary-app-app-m9uusrqo3ivt6w2kavnnsf.streamlit.app)

## 📋 Features
- **Job Market Analysis**: View top 15 job titles and category distributions
- **Salary Insights**: Compare salaries by education level, experience, and job category
- **Skills Analysis**: Discover most in-demand skills and their salary premiums
- **Interactive Filters**: Filter data by salary range, job category, education level, and experience level

## 🛠️ Tech Stack
- **Framework**: Streamlit
- **Data Processing**: Pandas
- **Visualization**: Matplotlib
- **Deployment**: Streamlit Cloud

## 📁 Project Structure
- app.py # Main application
- Salary_Data.xlsx # Dataset
- requirements.txt # Python dependencies
- README.md # Project documentation

## 🏃 How to Run Locally
1. **Clone the repository**
   ```bash
   git clone https://github.com/zziqi564-oss/mysalary-streamlit-app.git
   cd mysalary-streamlit-app
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the app**
   ```bash
   streamlit run app.py
   ```

## 📦 Dependencies
- streamlit >= 1.28.0
- pandas >= 2.0.0
- matplotlib >= 3.7.0
- openpyxl >= 3.1.0

## 🔑 Key Findings
- Software Engineering and Marketing are the two largest job categories
- Higher education provides the biggest salary boost at senior experience levels
- Technical skills like Cloud, API Design, and Machine Learning command the highest salaries
